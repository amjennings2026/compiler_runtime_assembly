Hello!

This is a project demonstrating how a compiler interacts with a runtime environment using assembly language. 
This repository explores how compiled programs execute at the machine level and how runtime components support program execution.

Project Goals:

Demonstrate how compiled programs interact with the runtime.

Explore assembly-level program structure.

Understand calling conventions and stack management.

Provide examples of runtime support routines.

Serve as an educational reference for compiler/runtime interaction.
